# QuickRent Backend

Vehicle rental platform backend API built with Node.js and Express.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Run development server:
```bash
npm run dev
```

## API Endpoints

- `GET /` - Welcome message
- `GET /health` - Health check
